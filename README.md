## ✨ What is okiosk?

**okiosk** is a full-featured e-commerce application built with Flutter, designed for seamless shopping experiences across mobile, web, and desktop. It combines a polished UI with robust backend integration and modern development practices.

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🤖 **AI Product Search** | Natural language search to find products quickly |
| 🎯 **AI Recommendations** | Personalized product suggestions based on your preferences |
| 📝 **AI Product Descriptions** | Rich, contextual product descriptions generated by AI |
| ⭐ **AI Product Reviews** | AI-powered review summaries and insights |
| 💬 **AI Product Q&A** | Get instant answers to your product questions |
| ⚖️ **AI Product Comparison** | Compare products by category, price range, and brand |
| 🛍️ **Product Catalog** | Browse products with categories, ratings, and dynamic product cards |
| 🛒 **Smart Cart** | Full cart management with coupons, quantity controls, and order tracking |
| 💳 **Checkout Flow** | Streamlined checkout with success screens and payment integration |
| 📱 **Cross-Platform** | Runs on iOS, Android, Web, and Windows desktop |
| ☁️ **Supabase Backend** | Real-time database, auth, and storage powered by Supabase |
| 🎤 **Voice-to-Text** | Voice input integration for hands-free interaction |
| 📷 **QR Support** | QR code generation and scanning capabilities |
| 🌐 **Network Aware** | Offline detection and connectivity status widgets |
| 🎨 **Polished UI** | Shimmer loaders, Lottie animations, custom themes |

---

## 🛠️ Tech Stack

```
Flutter  •  GetX (State)  •  Supabase  •  Firebase  •  AI/LLM  •  SQLite
```

| Layer | Technologies |
|-------|--------------|
| **Frontend** | Flutter, GetX, Lottie, Shimmer |
| **Backend** | Supabase, Firebase Core |
| **AI** | Product search, recommendations, descriptions, reviews, Q&A, comparison |
| **State** | GetX, GetStorage |
| **Networking** | HTTP, WebSocket, Connectivity Plus |
| **Storage** | Supabase Storage, SQLite (local) |


## 🏃 Getting Started

### Prerequisites

- Flutter SDK `^3.6.1`
- Dart `^3.6.1`

### Run the app

```bash
# Install dependencies
flutter pub get

# Run on your device
flutter run
```
